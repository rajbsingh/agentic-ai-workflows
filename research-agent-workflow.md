# Research Agent Workflow

This workflow describes a high-level agentic AI pattern for research, analysis, summarization, and report preparation.

The purpose of the Research Agent is to help users collect information, organize findings, identify gaps, and prepare structured outputs for human review.

---

## Workflow Objective

Support research-heavy tasks by using AI to collect, structure, summarize, and synthesize information.

The agent should help with:

- Topic understanding
- Source collection
- Information extraction
- Summary creation
- Gap identification
- Structured report generation
- Human review before final use

---

## Target Users

This workflow may support:

- Business leaders
- Consultants
- Startup founders
- Product teams
- Analysts
- Researchers
- Students
- Content creators
- Enterprise strategy teams

---

## Agent Responsibilities

The Research Agent may perform the following tasks:

1. Understand the research question
2. Break the topic into subtopics
3. Identify required information
4. Search trusted sources
5. Extract relevant points
6. Compare findings
7. Identify contradictions or uncertainty
8. Summarize insights
9. Highlight gaps
10. Prepare a structured report

---

## High-Level Workflow

```text
User provides research question
  ↓
Agent clarifies objective
  ↓
Agent breaks research into subtopics
  ↓
Agent searches trusted sources
  ↓
Agent extracts key findings
  ↓
Agent compares and organizes information
  ↓
Agent identifies gaps and uncertainty
  ↓
Agent prepares structured report
  ↓
Human reviews and finalizes output
